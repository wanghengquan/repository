===================================Readme For Altera=======================================
Options help:

job-dir: the resutl directory located, all cases are located in the directory
design: which design you want to scan

report-name: the report name if you specified,or the basename of the job directory will be used
report-path: the report path you want to store

these options can be used as lattice scan scripts.
---------------DEMO------------------

you can run it as:
python trunk\tools\scan_report\bin\run_scan_altera.py --job-dir=C:\Users\yzhao1\Desktop\ss\competor\alter\E60_cylone --design=08_vj1kfpga
python trunk\tools\scan_report\bin\run_scan_altera.py --job-dir=C:\Users\yzhao1\Desktop\ss\competor\alter\E60_cylone
